# 大和久古伝(やまときゅうこでん) ver.0.1 (β version)

## ストーリー

春の訪れとともに異変が起こる。
本来は散っていくはずの桜が 決して散らず、満月の夜に逆に咲き狂う ようになった。
花は美しく輝くほどに妖しく、里の者たちは「これは吉兆か凶兆か」と困惑する。

人々の噂では、
かつて「桜に封じられた霊」が目覚めたのではないか と囁かれていた。

巫女は、この異変を解決するため、
桜が咲き乱れる大和久の地を巡る旅に出る。

## 操作方法
* 矢印キーで上下左右移動
* 左Shiftで低速移動(当たり判定確認)
* Zキーで弾幕発出
* Xキーでスペルカード発動(一定時間無敵&範囲内の敵弾幕消去)
* メニュー画面にて，Enterで決定，Escで戻り
